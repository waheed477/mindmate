🚀 Quick Start for Recruiters
Prerequisites
Node.js 18+ installed

MongoDB installed and running locally (or use MongoDB Atlas)

Git installed

Step 1: Clone the Repository
bash
git clone https://github.com/waheed477/mindmate.git
cd mindmate
Step 2: Setup Backend
bash
cd backend
npm install
cp .env.example .env
# (Optional) Add your Gmail credentials in .env for real email functionality
npm run dev
Expected output:

text
✅ MongoDB connected successfully
📁 Collections: users, patients, doctors, appointments, messages, prescriptions
🚀 Backend API running on port 5000
🔌 Socket.io enabled
Step 3: Setup Frontend
Open a new terminal:

🐛 Troubleshooting
Issue	Solution
MongoDB connection error	Start MongoDB: mongod in terminal
Port 5000 already in use	Change PORT in backend/.env
Port 3000 already in use	Change VITE_PORT in frontend/client/.env
Registration fails	Check backend terminal for validation errors
Email not sending	Add valid Gmail credentials in .env
WebSocket errors	Ensure backend is running on port 5000
